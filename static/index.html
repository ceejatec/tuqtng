Couchbase Query Developer Preview 3

Welcome to Couchbase Query. The endpoints below are available at this engine.

Query REST API

You may post queries to /query. Results will be returned in response as a JSON data set.

Tutorial

You may visit the Query Tutorial at /tutorial for a brief introduction, and a quick way of running ad-hoc queries.
Note : start_tutorial.sh or start_tutorial.bat in case of Windows should be run in order to view the tutorial

Other Resources

For further resources, documentation, community pages, please visit query.couchbase.com
